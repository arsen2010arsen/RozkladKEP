# 🎓 RozkladKEP Bot

Telegram-бот для автоматичного парсингу та розсилки розкладу занять Коледжу електронних приладів ІФНТУНГ (КЕП). 
Створений для зручного доступу до актуального розкладу (особливо для групи ПІ-25-01 та інших).

## 🚀 Основний функціонал
* **Живий парсинг:** Дані беруться безпосередньо з офіційного сайту коледжу в реальному часі.
* **Автоматична розсилка:** Бот щодня (з Пн по Пт) о 00:00 автоматично надсилає розклад на наступний день усім підписаним користувачам.
* **Врахування тижнів:** Автоматичне визначення поточного тижня (чисельник/знаменник) відносно початку семестру.
* **Статистика користувачів:** Збереження юзерів та їхніх груп у хмарну базу даних MongoDB.

## 🛠 Технології
* **Python 3.12+**
* **aiogram 3.x** — асинхронний фреймворк для Telegram API.
* **aiohttp** — для вебсервера (Render) та асинхронних запитів.
* **BeautifulSoup4 / re** — для веб-скрапінгу (парсингу HTML та JS-масивів).
* **Motor (MongoDB)** — асинхронна робота з базою даних користувачів.
* **APScheduler** — для налаштування cron-задач (щоденної розсилки).

## ⚙️ Налаштування та Запуск
Бот оптимізований для безкоштовного хостингу на **Render**. Для роботи потрібні наступні Environment Variables:
* `BOT_TOKEN` — токен бота від @BotFather.
* `MONGO_URL` — рядок підключення до MongoDB Atlas.
* `ADMIN_ID` — цифровий ID адміна (для доступу до команди `/users`).
* `PORT` — порт вебсервера (Render задає його автоматично).
